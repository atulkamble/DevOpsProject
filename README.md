# DevOpsProject
Hosting website using terraform, awscli &amp; git

Prerequisites: git, terraform, aws-cli

// Step 0
```
terraform -v
```
```
aws --version
```
// on instance 
```
git --version
```
// Step 1 | Navigate to Project Directory

// Step 2 | Run terraform files
```
terraform init
terraform validate
terraform fmt
terraform plan
terraform apply
terraform destroy
```
